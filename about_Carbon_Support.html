about_Carbon_Support

This topic describes how you can get help using Carbon.

Overview

Remember, Carbon is open-source software, and comes "as is", without warranties or conditions of any kind, either express or implied. The Carbon users and developers are volunteering to help you. Please respect their time and assistance.

Chat

Carbon has a Gitter chat room you can use to ask questions and get help. The project owner/maintainer usually hangs out there during business hours, Monday through Friday. He's in the Pacific time zone. You'll need a GitHub account to login and ask questions.

E-Mail

Carbon has a mailing list: carbon@freelists.org. You don't need to subscribe to ask questions, but you do need to subscribe to be part of the community and answer questions. Subscribe to the list..

Reporting Bugs And Requesting New Features And Enhancements

If you've found a bug or want to request an enhancement, create an issue on the project's Bitbucket site.
